# 九、下一步建议

### 开发阶段: 第 1 步 → 配外部 API Key

| 优先级 | 服务 | 变量 | 获取方式 |
|--------|------|------|---------|
| 🔴 | LLM | QWEN3_API_URL + QWEN3_API_KEY | 阿里云百炼平台 |
| 🔴 | Embedding | EMBEDDING_API_URL + EMBEDDING_API_KEY | 同上 |
| 🟡 | 微信登录 | WX_APP_ID + WX_APP_SECRET | 微信公众平台 |
| 🟡 | COS | COS_SECRET_ID + COS_SECRET_KEY | 腾讯云控制台 |
| 🟢 | OCR | PADDLEOCR_URL | 部署 PaddleOCR 容器 |
| 🟢 | 微信支付 | WX_MCH_ID + WX_API_V3_KEY | 微信商户平台 |

### 开发阶段: 第 2 步 → 补 P1 前端页面

2 个缺失页面 + 2 个页面增强，约 2.5 天工作量。

### 开发阶段: 第 3 步 → 部署基础设施

Docker Compose 编排 + PostgreSQL/pgvector + Redis + HTTPS，约 3.5 天工作量。

### 开发阶段: 第 4 步 → 验收测试

AI 组卷成功率、OCR 识别率、导出成功率、支付成功率四项指标测试。

> **结论**：代码框架完成度 72%，Feature V1.0 范围 95% 可运行（Dev 模式）。阻塞上线的核心问题不是代码，而是 **6 个外部服务的 API key/地址配置** 和 **生产基础设施部署**。代码侧剩余工作约 10 天（微信支付 V3 完整对接 3d + Python 导出服务 2d + Docker/部署 3.5d + 前端补完 2.5d）。
