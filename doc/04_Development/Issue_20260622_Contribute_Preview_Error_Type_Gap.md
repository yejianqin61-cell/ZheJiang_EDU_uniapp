# Issue: 题库贡献预览页提交异常仍在使用弱错误类型

日期：2026-06-22

## 现象

- `frontend-web/src/pages/contribute/preview/index.vue` 的提交审核分支仍在使用 `catch (error: any)`
- 页面已经复用共享 `contribution` API 模块，但异常分支没有同步收口

## 风险

- 题库贡献提审链路的异常对象继续绕开类型系统。
- 后续如果扩展重试、失败提示或审核前校验，弱类型分支会继续扩散。

## 处理建议

1. 使用 `unknown` 接住提审异常
2. 在页面内收口最小错误消息提取逻辑
3. 保持原有预览、提审成功跳转和失败提示行为不变
