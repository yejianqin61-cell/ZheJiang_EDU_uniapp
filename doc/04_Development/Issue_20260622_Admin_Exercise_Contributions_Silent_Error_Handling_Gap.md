# Issue: 管理端练习审核页多条操作链路仍存在静默失败

日期：2026-06-22

## 现象

- `frontend-web/src/pages/admin/exercise-contributions/index.vue` 中单条通过、单条拒绝和批量审核分支仍在使用空 `catch`
- 用户取消弹窗与接口真实失败没有区分，失败时页面缺少明确反馈

## 风险

- 管理端练习审核链路失败时容易静默，影响审核人员判断操作是否真正落地。
- 后续扩展审核原因、批量流程和异常处理时，静默失败分支会继续扩散。

## 处理建议

1. 区分弹窗取消与接口失败
2. 对真实失败补充明确错误提示
3. 保持原有成功提示与列表刷新行为不变
