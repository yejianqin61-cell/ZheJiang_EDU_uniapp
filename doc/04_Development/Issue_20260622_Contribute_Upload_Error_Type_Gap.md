# Issue: 题库贡献上传页提交异常仍在使用弱错误类型

日期：2026-06-22

## 现象

- `frontend-web/src/pages/contribute/upload/index.vue` 的上传提交分支仍在使用 `catch (error: any)`
- 页面已经复用共享 `contribution` API 模块，但异常分支没有同步收口

## 风险

- 题库贡献上传链路的异常对象继续绕开类型系统。
- 后续如果扩展上传重试、失败提示或多文件能力，弱类型分支会继续扩散。

## 处理建议

1. 使用 `unknown` 接住上传异常
2. 在页面内收口最小错误消息提取逻辑
3. 保持原有上传进度、成功跳转和失败提示行为不变
