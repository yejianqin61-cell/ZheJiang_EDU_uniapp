# Issue: 练习类目页加载失败仍存在静默与弱类型分支

日期：2026-06-22

## 现象

- `frontend-web/src/pages/exercises/category.vue` 原实现使用压缩脚本和裸 `any[]`
- 类目加载失败与同步课时加载失败分支缺少统一错误消息提取，页面状态清理也不够明确

## 风险

- 练习浏览入口属于甲方可直接触达的用户链路，加载失败时如果状态残留或提示不一致，容易误导用户判断
- 页面内继续保留弱类型数据结构，会降低后续练习模块维护和回归测试的稳定性

## 处理建议

1. 为类目和课时加载失败补充统一错误消息提取
2. 失败时显式清空当前列表或课时分组，避免旧态残留
3. 补充页面回归测试，覆盖类目失败与同步课时失败分支
