# Issue: 练习试卷上传页仍存在静默失败分支

日期：2026-06-22

## 现象

- `frontend-web/src/pages/contribute/exercise-upload/index.vue` 在加载类目、加载课时和提交上传时仍存在空 `catch` 或吞错分支
- 当接口失败时，页面会静默回退为空列表，或者完全依赖外层拦截器，用户无法在当前上下文获得明确反馈

## 风险

- 上传页属于教师贡献主链路，静默失败会直接影响试卷上传完成率
- 类目、课时和上传错误若没有页面内反馈，用户很难判断是数据为空、选择有误还是接口异常

## 处理建议

1. 为类目加载、课时加载和上传失败补充明确错误提示
2. 保留现有空列表兜底和提交流程，不中断页面交互
3. 增加页面回归测试，覆盖三个失败分支的提示行为
