# Issue: 练习首页标签切换仍在使用弱类型断言

日期：2026-06-22

## 现象

- `frontend-web/src/pages/exercises/index.vue` 模板点击事件仍在使用 `tab = t.key as any`
- 页面脚本没有把练习入口标签定义为显式共享联合类型，局部状态容易继续变宽

## 风险

- 练习入口页标签状态继续绕开类型系统。
- 后续如果扩展入口标签、跳转参数或筛选逻辑，弱断言容易掩盖分支遗漏。

## 处理建议

1. 为练习入口标签补齐显式联合类型
2. 使用 typed handler 处理标签切换
3. 保持原有入口展示和跳转行为不变
