# Issue: 管理端仪表盘首屏加载仍存在静默失败

日期：2026-06-22

## 现象

- `frontend-web/src/pages/admin/dashboard/index.vue` 在加载仪表盘统计时仍使用空 `catch`
- 当仪表盘接口失败时，页面会静默回退到默认 0 值，用户无法区分“暂无数据”和“接口异常”

## 风险

- 管理端仪表盘属于运营入口页，静默失败会误导运营判断当前题库、订单和审核积压情况
- 接口异常被默认值掩盖后，问题更难在首屏被及时发现

## 处理建议

1. 为仪表盘统计加载失败补充明确错误提示
2. 保留默认值兜底与图表初始化，避免页面渲染中断
3. 补充页面回归测试，覆盖首屏加载失败提示分支
