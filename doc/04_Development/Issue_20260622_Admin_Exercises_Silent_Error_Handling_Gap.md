# Issue: 管理端练习管理页多条操作链路仍存在静默失败

日期：2026-06-22

## 现象

- `frontend-web/src/pages/admin/exercises/index.vue` 中删除分类、创建课时、删除课时、删除试卷等分支仍在使用空 `catch`
- 用户取消弹窗和接口真实失败没有区分，失败时页面缺少明确反馈

## 风险

- 管理端练习管理链路失败时容易静默，影响运营排障和内容维护效率。
- 后续扩展分类/课时/试卷操作时，静默失败分支会继续扩散。

## 处理建议

1. 区分弹窗取消与接口失败
2. 对真实失败补充明确错误提示
3. 保持原有成功提示与刷新行为不变
