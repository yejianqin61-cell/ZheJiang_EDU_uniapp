# 微信小程序注册 —— 手把手操作指南

Version 1.0 | 2026-06-11

---

## 注册前准备

请确认以下材料都在手边：

| # | 材料 | 说明 |
|---|------|------|
| 1 | 营业执照原件（照片/扫描件） | 清晰、四角完整、在有效期内 |
| 2 | 法定代表人身份证正反面 | 有效期内的二代身份证 |
| 3 | 法定代表人本人微信 | **非常重要：必须是营业执照上那个法人本人的微信，且已实名认证** |
| 4 | 一个没注册过微信公众平台的邮箱 | QQ邮箱/Gmail/163 都行 |
| 5 | 手机号 | 能收到验证码 |
| 6 | 对公账户信息（企业）或法人银行卡（个体户） | 后续申请微信支付用 |

> ⚠️ **最关键的坑**：微信验证会比对工商数据库。如果营业执照刚办下来（<5天），工商数据未同步，会报"企业信息或法定代表人信息不一致"。**建议营业执照办好后等 5 个工作日再注册。**

浏览器打开 **https://mp.weixin.qq.com**

点右上角 **「立即注册」**：

```

│                             │
│   [ 订阅号 ]   [ 服务号 ]   │
│   [ 小程序 ]   [ 企业微信 ]  │
│                             │
│     👆 选「小程序」           │
└─────────────────────────────┘
```

**关键选择**：

| 主体类型 | 能用微信支付吗 | 需要什么 |
|----------|:---:|------|
|  |  |  |
| **个体工商户** | ✅ | 营业执照 + 法人身份证 + 法人银行卡 |
| **个人** | ❌ **不能！** | 身份证就行 |

> 🔴 **千万不要选个人！** 个人主体不支持微信支付，等于白注册。

## Step 5：法人微信扫码验证

最关键的一步。页面会弹出一个二维码。

**用营业执照上法定代表人的本人微信扫码**：

1. 法人打开微信扫一扫
2. 扫码后进入微信实名认证
3. 微信核对法人姓名 + 身份证号
4. 人脸识别验证
5. 验证通过 → 注册提交

> 🔴 **如果这一步报"企业信息或法定代表人信息不一致"**：
> - 检查营业执照是否刚办（<5天）→ 等5天
> - 检查微信实名是否法人本人 → 微信→我→服务→右上角→实名认证
> - 去 https://www.gsxt.gov.cn 核对执照信息是否有变更
> - 注意：企业有法人变更记录的，工商数据更新需要 5-15 天

## Step 6：注册成功 → 获取 AppID

审核通过后（即时~7天），登录 https://mp.weixin.qq.com：

```
左侧菜单 → 开发 → 开发管理 → 开发设置
```

找到这两个东西：

```
┌────────────────────────────────────────┐
│                                        │
│   AppID（小程序ID）                     │
│   wx_________________________________  │
│                                        │

│   点击「生成」→ 管理员扫码               │
│   _________________________________    │
```

## 常见问题

**Q：注册要多久？**
A：法人微信扫码秒过的话，当天就能拿到 AppID。如果触发工商信息人工审核，1-7 个工作日。

**Q：审核期间小程序能用吗？**
A：不能。但你可以用 AppID 先在 UniApp 里配置编译，等审核通过后上传代码即可。

**Q：个人主体以后能改成企业吗？**
A：**不能。** 注册后主体不可更改。所以一开始就不要选个人。

**Q：没有对公账户怎么办？**
A：选个体工商户主体，用法人的个人银行卡。微信支付支持个体工商户的法人银行卡收款。

---

> 📅 建议时间线：今天注册小程序 → 等审核同时去申请微信支付商户号 → 备案通过当天全部就绪。
