# 第一件事：买云服务器

## 1.1 打开阿里云

用电脑浏览器打开：**https://www.aliyun.com**

如果还没有阿里云账号，点右上角"免费注册"：
- 输入手机号
- 接收验证码
- 设置登录密码
- 如果有支付宝，用支付宝扫码实名认证更快

## 1.2 进入 ECS 购买页

登录后：
1. 顶部搜索框输入 **"ECS"**，点搜索
2. 点击 **"云服务器 ECS"**
3. 点击橙色按钮 **"创建实例"**

## 1.4 确认订单

1. 页面右侧看价格预览：按量付费约 **0.3-0.5 元/小时**
2. 勾选 ☑ **"我已阅读并同意《云服务器ECS服务条款》"**
3. 点击橙色 **"创建实例"**
4. 等待约 1-2 分钟，状态变为 **"运行中"**

## 1.5 获取服务器 IP

实例创建成功后：
1. 点击实例 ID，进入详情页
2. 找到 **"公网 IP"** 一栏（格式如 `47.96.123.456`）
3. **复制这个 IP 地址**，这是您服务器的大门

> 如果选择包月（推荐）：回到实例列表 → 点击实例右侧"更多"→ "转包月"→ 选 1 个月

# 第二件事：申请 AI Key

## 2.1 打开阿里云百炼

用电脑浏览器打开：**https://bailian.console.aliyun.com**

如果提示登录，用和上面同一个阿里云账号登录。

## 2.2 开通服务（首次使用）

首次进入百炼控制台，页面会自动引导您开通服务：

1. 看到"立即开通"按钮 → 点击
2. 勾选服务协议 → 确认开通
3. 等待几秒，显示"开通成功"

## 2.3 创建 API Key

1. 鼠标移到页面**右上角头像**
2. 点击 **"API-KEY 管理"**
3. 点击蓝色按钮 **"创建 API-KEY"**
4. 弹出确认框 → 点击 **"确定"**
5. 页面显示一串字符，格式如：`sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

```
┌─────────────────────────────────────────────┐
│                                              │
│   sk-85819fdd5c7f45a29baae7bd3156af5a        │
│         ↑ 类似这样的格式，复制它              │
│                                              │
│ ⚠️ 这串字符只显示一次！立刻复制保存！          │
│ ⚠️ 不要泄露给其他人，这是您的AI调用凭证       │
└─────────────────────────────────────────────┘
```

# 常见问题

**问：我没有阿里云账号怎么办？**
答：用手机号注册一个，5 分钟。支付宝实名认证更快。

**问：服务器密码忘了怎么办？**
答：阿里云控制台可以重置密码——实例详情 → 重置密码。

**问：这个费用谁出？**
答：服务器和 AI 调用费用由甲方承担。开发部署费用已包含在项目款内。

**问：后续怎么从小程序打开？**
答：等域名备案完成后，再部署微信小程序版本。今天先让老师们用手机浏览器访问。
