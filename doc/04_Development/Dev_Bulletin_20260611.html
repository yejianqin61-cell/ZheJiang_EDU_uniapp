今晚踩坑 & 修复清单

🐛 #1 PM2 Cluster × SQLite 致命竞态

2进程各自内存隔离，A进程写 B进程读不到 → "试卷不存在"。修复：instances=1, exec_mode=fork

🐛 #2 百炼 API 欠费 Arrearage

组卷返回 500，日志 "Access denied, account not in good standing"。修复：充值余额恢复

🐛 #3 python-docx RGBColor API 变更

新版 python-docx 移除了 RGBColor，__import__ 调用失效 → 导出 500。修复：import 到顶并 gunicorn 重启

🐛 #4 COS 占位值被当真实配置

.env 中 COS_SECRET_ID=your_secret_id 被判断为已配置 → 真调腾讯云报 bucket 不存在。修复：清空占位值

🐛 #5 EXPORT_SERVICE_URL 指向 Docker 容器名

.env 残留 http://export:5000 → 后端调不通 Python。修复：改 localhost:5000

🐛 #6 全项目 5 处 localhost:3000 硬编码

payment/index.vue · orders/index.vue · orders/detail/index.vue · admin/orders/index.vue(×2)。手机 localhost 指向自身。修复：统一收敛到 config/env.ts

🐛 #7 nginx 缺少 /v1/download/ 路由

导出返回 /v1/download/xxx.docx，nginx 未转发 → 404。修复：补 location 块

🐛 #8 Docker Hub 被墙

docker-compose 拉镜像 SSL connection reset。放弃 Docker，改用 PM2 + Node.js 直装。Ubuntu 26.04 过新加剧兼容问题
