# Issue: 打印下单页仍存在静默失败分支

日期：2026-06-22

## 现象

- `frontend-web/src/pages/print/checkout/index.vue` 在加载打印定价和收货地址时仍存在空 `catch`
- 当接口失败时，页面会静默回退为空定价或空地址列表，用户无法判断是数据为空还是请求异常

## 风险

- 打印下单页属于付费前关键链路，静默失败会误导用户对价格和可选地址状态的判断
- 如果创建订单报错信息结构不完整，前端回退提示不稳定，后续排查成本会上升

## 处理建议

1. 为打印定价和收货地址加载失败补充明确错误提示
2. 保持页面默认值兜底，避免渲染中断
3. 增加页面回归测试，覆盖加载失败和订单创建失败兜底提示
