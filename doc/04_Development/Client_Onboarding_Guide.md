## 一、微信小程序 AppID + AppSecret

### 1.1 注册小程序账号

```
1. 打开 https://mp.weixin.qq.com
2. 点击右上角 "立即注册"
3. 选择 "小程序"
4. 填写邮箱（未注册过微信公众平台的邮箱）
5. 激活邮箱 → 选择 "个人" 或 "企业" 主体
6. 填写身份信息（个人：身份证；企业：营业执照）
7. 管理员微信扫码验证
```

> **注意**：个人主体无法使用微信支付。如需微信支付，必须选择**企业/个体工商户**主体。

### 1.2 获取 AppID 和 AppSecret

```
1. 登录 https://mp.weixin.qq.com
2. 左侧菜单 → 开发 → 开发管理
3. 点击 "开发设置" 标签
4. 找到 AppID（小程序ID）→ 复制
5. 找到 AppSecret（小程序密钥）→ 点击 "生成"（需管理员扫码）
6. 将这两个值发给我们
```

> ⚠️ **AppSecret 只显示一次，务必复制保存！**

### 1.3 配置服务器域名白名单

```
在同一个页面往下滚 → "服务器域名" → 点击 "修改"
  request合法域名：  https://你的域名
  uploadFile合法域名：https://你的域名
  downloadFile合法域名：https://你的域名
```

> 注：域名准备好后再配，不需要着急。

## 二、微信支付商户号

### 2.1 申请商户号

```
1. 打开 https://pay.weixin.qq.com
2. 点击 "接入微信支付" → "马上注册"
3. 用小程序的管理员微信扫码
4. 选择主体类型（企业/个体工商户）
5. 上传营业执照
6. 填写法人信息 + 对公账户信息
7. 微信支付会向对公账户打一笔小额验证款（0.01-1元）
8. 查询对公账户收到的金额，回填验证
9. 审核通过（1-3 个工作日）
```

### 2.2 获取商户号和 API 密钥

审核通过后：
```
1. 登录 https://pay.weixin.qq.com
2. 账户中心 → 商户信息 → 复制 "商户号 (MCH_ID)"
3. 账户中心 → API 安全 → 设置 APIv3 密钥
   → 随机生成 32 位字符串 → 复制保存
```

### 2.3 申请 API 证书

```
1. 账户中心 → API 安全 → 申请 API 证书
2. 下载 "微信支付商户API证书申请工具"
3. 按工具提示生成 CSR 并提交
4. 下载证书压缩包，解压得到三个文件：
   - apiclient_key.pem    → 这就是 "商户私钥"（用记事本打开复制全部内容）
   - apiclient_cert.pem   → 商户证书（备用）
   - 证书序列号            → 在商户平台 API 安全页面查看，40位十六进制字符串
```

> 把以下信息发给我们：
> - 商户号 (MCH_ID)
> - API V3 密钥 (32位随机字符串)
> - 证书序列号
> - apiclient_key.pem 文件内容（私钥）

## 三、域名 + 备案

### 3.1 购买域名

```
1. 打开 https://wanwang.aliyun.com（阿里云万网）
   或 https://dnspod.cloud.tencent.com（腾讯云 DNSPod）
2. 搜索你想用的域名，比如：
   - zjai-paper.com
   - zjjiaoshi.com
   - ai-zujuan.cn
3. 加入购物车 → 结算（.com约60元/年，.cn约30元/年）
4. 完成实名认证（上传身份证）
```

### 3.2 域名备案（必须！）

```
1. 登录阿里云/腾讯云控制台 → 搜索 "备案"
2. 点击 "开始备案" → 填写网站信息
   网站名称：AI智能组卷（或：浙江省教师组卷平台）
   网站内容：教育考试组卷服务
3. 上传身份证正反面 + 手持身份证照片
4. 下载 "备案真实性核验单"，签字后拍照上传
5. 服务商初审（1-2 天）→ 管局终审（3-15 天）
```

> ⚠️ **域名必须备案才能在国内服务器使用！没有备案 = 无法上线！**

### 3.3 DNS 解析

备案通过后：
```
1. 云控制台 → 域名解析 → 添加记录
2. 记录类型：A
3. 主机记录：@（直接解析）和 www
4. 记录值：你的云服务器公网 IP 地址
5. 保存
```

## 四、云服务器

### 4.1 购买服务器

```
1. 打开 https://ecs.console.aliyun.com（阿里云 ECS）
   或 https://console.cloud.tencent.com/cvm（腾讯云 CVM）
2. 点击 "创建实例"
3. 选择配置：
   ┌─────────────────────────────┐
   │ 地域：     离浙江最近的（如华东1-杭州）│
   │ 实例规格：  2 vCPU / 4 GiB         │
   │ 系统镜像：  Ubuntu 22.04 或 CentOS 7 │
   │ 系统盘：    40 GB                  │
   │ 带宽：      3-5 Mbps              │
   │ 购买时长：  1 个月起               │
   └─────────────────────────────┘
4. 设置 root 密码（务必记好！）
5. 提交订单
```

> 推荐配置：2核4G，约 100-150 元/月。先买 1 个月。

### 4.2 开放端口

```
服务器创建好后：
1. 进入实例详情 → 安全组 → 配置规则
2. 确保以下端口已开放：
   - 22   (SSH 远程登录)
   - 80   (HTTP)
   - 443  (HTTPS)
3. 不是必须的端口不要开
```

### 4.3 需要提供给我们

```
- 服务器公网 IP 地址
- root 密码（或给我们添加一个 SSH 密钥）
```

## 五、阿里云百炼 API Key（AI 大模型）

组卷用到的大模型服务，按量付费，很便宜。

### 5.1 开通百炼

```
1. 打开 https://bailian.console.aliyun.com
2. 用阿里云账号登录（没有就先注册）
3. 首次进入会自动开通服务
4. 左侧菜单 → 模型广场 → 找到通义千问系列
5. 点击 "API 调用" → 系统会提示开通 DashScope
6. 同意协议 → 开通
```

### 5.2 获取 API Key

```
1. 百炼控制台 → 右上角头像 → API-KEY 管理
2. 点击 "创建 API-KEY"
3. 复制生成的 Key（格式：sk-xxxxxxxxxxxxx）
```

### 5.3 费用预估

```
组卷一次消耗约 2,000-3,000 tokens
价格约 0.002-0.004 元/次
1000 次组卷 ≈ 2-4 元
100 元预算够 25,000 次组卷
```
