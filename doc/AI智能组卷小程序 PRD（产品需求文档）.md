# AI智能组卷小程序

## Product Requirements Document (PRD)

Version 1.0

------

# 1. 产品概述

## 1.1 产品名称

AI智能组卷小程序

------

## 1.2 产品定位

面向浙江省中小学教师的AI智能组卷平台。

通过知识库检索与大模型生成能力，实现：

教师输入需求 → AI自动组卷 → 在线预览 → 微信支付 → 导出试卷

完整业务闭环。

------

## 1.3 产品目标

### 用户目标

帮助教师在30秒内完成试卷生成。

### 商业目标

- 建立高质量题库资产
- 实现按次付费变现
- 降低教师备课成本
- 提高试卷生成效率

------

# 7. AI知识库设计

## 知识点中心

由AI自动维护。

------

知识点生成流程：

```
题目

↓

AI识别知识点

↓

知识点库搜索

↓

找到

→ 直接关联

↓

未找到

→ 自动创建

↓

自动归类

↓

保存
```

------

知识点标准化规则：

使用Embedding相似度匹配。

若知识点语义相近：

自动归并。

避免：

一次函数

线性函数

初等函数

等重复知识点出现。
