| F-005 | 难度选择 | AI组卷 | **P0** | 四选一：简单/中等/困难/混合。混合模式按难度均匀分布。 |

| F-008 | 试卷预览 | 试卷 | **P0** | 卡片式展示题目列表，含题型/内容/选项/分值。答案与解析默认折叠。 |
| F-009 | 试卷重新生成 | AI组卷 | P1 | 对当前组卷条件不满意，一键重新生成（不消耗额外次数，每日限制3次）。 |
| F-010 | 微信支付 | 支付 | **P0** | 唤起微信支付收银台，按次付费。30分钟未支付订单自动取消。 |
| F-011 | DOCX导出 | 导出 | **P0** | A4排版，含页眉页脚，答案独立分页。宋体正文，黑体标题。 |

| F-013 | 历史订单列表 | 订单 | P1 | 按时间倒序展示历史订单，显示日期/科目/题量/金额/状态。 |
| F-014 | 订单筛选 | 订单 | P2 | 按科目、时间范围筛选历史订单。 |

| F-016 | 个人中心 | 用户系统 | P1 | 展示用户头像、昵称、已生成次数、剩余次数（若有）。 |

## 2. 管理后台功能 (Admin)

### 2.1 资料入库

| ID | Feature | Module | Priority | Description |
|----|---------|--------|----------|-------------|
| F-101 | 文件上传 | 资料入库 | **P0** | 支持DOC/DOCX/MD/PDF/PNG/JPG/JPEG。自动识别文件类型，路由至对应处理流程。文本类≤50MB，图片类≤10MB。 |
| F-102 | 上传进度展示 | 资料入库 | P1 | 实时显示上传与处理进度（上传 → OCR处理 → AI解析 → 待审核）。 |
| F-103 | OCR文字识别 | 资料入库 | **P0** | PaddleOCR引擎识别图片/扫描PDF文字，输出文本。处理时间≤60秒/文件。 |
| F-104 | AI题目切分 | 资料入库 | **P0** | 对OCR/文本提取结果进行题目边界切分，拆分为单题。 |
| F-105 | AI答案解析 | 资料入库 | **P0** | 识别题目类型、答案、解析，结构化存储。 |
| F-106 | AI知识点识别 | 资料入库 | **P0** | 分析题目内容 → Embedding检索知识点库 → 匹配则关联 / 未匹配则自动创建。 |
| F-107 | AI难度识别 | 资料入库 | **P0** | 自动判定难度等级：简单(1) / 中等(2) / 困难(3)。 |
| F-108 | 知识点自动去重归并 | 知识库 | **P0** | Embedding相似度≥0.92的知识点自动归并，防止语义重复。 |
